# Potato Planner

A cozy, comforting iOS productivity app that combines a simple time-based daily planner with a <i>Stephania erecta</i> plant (aka potato plant) progression system to make focusing feel rewarding.

## Features
- **Daily tasks**: add tasks with a title, description, and time estimate
- **Focus sessions**: start a focus timer tied to a task
- **Potato progression**: earn fertilizer and spuds from focused time to grow potatoes and buy new ones
- **Calendar view**: browse tasks by date and plan ahead

## Implementation
- **Swift + SwiftUI**
- **SwiftData**
- **MVVM architecture**
